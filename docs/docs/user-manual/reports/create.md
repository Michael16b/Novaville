# Créer un signalement

Apprenez à créer un signalement pour rapporter un problème dans votre commune.

## Qu'est-ce qu'un signalement ?

Un signalement permet d'**alerter votre administration locale** sur un problème rencontré dans votre ville :

- 🚧 Nid-de-poule sur la route
- 💡 Éclairage public défectueux
- 🗑️ Dépôt sauvage d'ordures
- 🚏 Mobilier urbain endommagé
- 🌳 Arbre dangereux
- Et bien plus...

## Créer un signalement (méthode rapide)

### 1. Accéder à la création

Deux façons d'accéder à la création :

**Option A** : Depuis l'accueil
- Ouvrez l'application
- Appuyez sur le grand bouton **"➕ Signaler"**

**Option B** : Depuis la carte
- Allez dans l'onglet **"Carte"** 🗺️
- Appuyez longuement sur le lieu du problème
- Sélectionnez **"Créer un signalement ici"**

### 2. Prendre une photo

La photo est **essentielle** pour bien documenter le problème.

1. Appuyez sur l'icône **📷 "Ajouter une photo"**
2. Choisissez :
   - **"Prendre une photo"** : Ouvrir l'appareil photo
   - **"Choisir dans la galerie"** : Sélectionner une photo existante
3. Prenez ou sélectionnez **1 à 5 photos**

> 💡 **Astuce** : Prenez plusieurs angles pour mieux montrer le problème.

### 3. Localiser le problème

L'application détecte automatiquement votre position actuelle.

**Si la position est correcte** :
- Rien à faire, passez à l'étape suivante ✅

**Si vous devez modifier la position** :
1. Appuyez sur **"Modifier la localisation"** 📍
2. Déplacez la carte pour pointer le bon endroit
3. Ou utilisez la barre de recherche pour trouver l'adresse
4. Appuyez sur **"Confirmer"**

### 4. Choisir une catégorie

Sélectionnez la catégorie qui correspond le mieux à votre problème :

- 🚧 **Voirie** : Routes, trottoirs, nids-de-poule
- 💡 **Éclairage** : Lampadaires, éclairage public
- 🗑️ **Propreté** : Déchets, dépôts sauvages
- 🚏 **Mobilier urbain** : Bancs, arrêts de bus
- 🌳 **Espaces verts** : Parcs, arbres
- 💧 **Eau** : Fuites, inondations
- 🚦 **Signalisation** : Panneaux, marquage au sol
- 🏗️ **Bâtiments** : Bâtiments municipaux
- 🔊 **Nuisances** : Bruit, pollution
- ➕ **Autre** : Si aucune catégorie ne convient

### 5. Décrire le problème

1. **Titre** (requis) : Résumez en quelques mots
   - ✅ Bon : "Nid-de-poule dangereux avenue Victor Hugo"
   - ❌ Éviter : "Problème"

2. **Description** (requis) : Détaillez le problème
   - Quoi : Décrivez précisément le problème
   - Où : Précisez l'emplacement exact si nécessaire
   - Quand : Depuis quand le problème existe-t-il ?
   - Impact : Comment ça vous affecte ?

**Exemple** :
```
Titre: Nid-de-poule dangereux avenue Victor Hugo

Description:
Un grand nid-de-poule s'est formé sur l'avenue Victor Hugo,
à 50 mètres après le carrefour avec la rue des Lilas.
Il fait environ 40 cm de diamètre et 10 cm de profondeur.
Présent depuis environ 2 semaines et s'agrandit avec la pluie.
Très dangereux pour les deux-roues.
```

### 6. Envoyer le signalement

1. Vérifiez que toutes les informations sont correctes
2. Appuyez sur **"Envoyer le signalement"** 📤

Un message de confirmation apparaîtra : **"Signalement envoyé avec succès !"** ✅

## Que se passe-t-il après ?

### 1. Validation (24-48h)

Votre signalement est **vérifié par un modérateur** pour s'assurer qu'il respecte les règles.

Statut : **🟡 En attente**

### 2. Attribution

Le signalement est **attribué à un agent** compétent.

Statut : **🔵 Confirmé**

Vous recevez une notification : *"Votre signalement a été pris en compte"*

### 3. Traitement

Les services compétents **interviennent** pour résoudre le problème.

Statut : **🟠 En cours**

Vous recevez une notification : *"Les travaux ont commencé"*

### 4. Résolution

Le problème est **résolu** !

Statut : **🟢 Résolu**

Vous recevez une notification : *"Votre signalement a été résolu"*

## Suivre vos signalements

### Voir tous vos signalements

1. Allez dans **Profil** 👤
2. Appuyez sur **"Mes signalements"** 📊
3. Vous verrez la liste de tous vos signalements avec leur statut

### Voir un signalement spécifique

1. Appuyez sur le signalement dans la liste
2. Vous verrez :
   - Les photos
   - La description complète
   - Le statut actuel
   - L'historique des changements
   - Les commentaires (si disponibles)

### Recevoir des notifications

Activez les notifications pour être alerté des changements :

1. **Profil** > **Paramètres** ⚙️
2. **Notifications**
3. Activez **"Signalements"**

## Modifier un signalement

Vous pouvez modifier votre signalement **avant qu'il ne soit confirmé** :

1. Allez dans **"Mes signalements"**
2. Appuyez sur le signalement à modifier
3. Appuyez sur **"Modifier"** ✏️
4. Faites vos modifications
5. Appuyez sur **"Enregistrer"**

> ℹ️ Une fois confirmé par un modérateur, vous ne pouvez plus modifier le signalement.

## Supprimer un signalement

Pour supprimer un signalement :

1. Allez dans **"Mes signalements"**
2. Appuyez sur le signalement
3. Appuyez sur le menu **⋮** (trois points)
4. Sélectionnez **"Supprimer"**
5. Confirmez la suppression

> ⚠️ La suppression est définitive et ne peut pas être annulée.

## Bonnes pratiques

### ✅ À faire

- **Prenez des photos claires** et bien cadrées
- **Soyez précis** dans la localisation
- **Décrivez clairement** le problème
- **Restez factuel** et objectif
- **Vérifiez** que le signalement n'existe pas déjà

### ❌ À éviter

- Photos floues ou de mauvaise qualité
- Descriptions vagues ou trop courtes
- Langage inapproprié ou insultant
- Signalements en double
- Informations personnelles sensibles

## Règles à respecter

Pour que votre signalement soit traité, il doit :

- ✅ Concerner un **problème réel** sur l'espace public
- ✅ Être **accompagné d'au moins une photo**
- ✅ Avoir une **description claire**
- ✅ Respecter les **règles de bienséance**
- ❌ Ne pas contenir d'informations personnelles de tiers
- ❌ Ne pas être diffamatoire ou injurieux

## Cas particuliers

### Urgence

Pour une **situation d'urgence** (danger imminent, accident) :

🚨 **N'utilisez PAS l'application**

Appelez immédiatement les services d'urgence :
- **SAMU : 15**
- **Police : 17**
- **Pompiers : 18**
- **Numéro d'urgence européen : 112**

### Signalement anonyme

Tous les signalements sont **publics** mais :
- Votre **nom complet n'apparaît pas** publiquement
- Seuls vos **initiales** sont visibles
- Les **agents municipaux** peuvent voir vos informations complètes

### Signalement rejeté

Si votre signalement est rejeté, vous recevrez une notification avec la raison.

Raisons courantes de rejet :
- Ne concerne pas l'espace public
- Doublon d'un signalement existant
- Photos inappropriées
- Description insuffisante
- Ne respecte pas les règles

Vous pouvez créer un nouveau signalement en tenant compte des remarques.

## Exemples de bons signalements

### Exemple 1 : Voirie

**Titre** : Trou dans la chaussée rue de la République

**Photos** : 3 photos montrant le trou sous différents angles

**Description** :
```
Trou important dans la chaussée rue de la République,
au niveau du n°45, côté impair.
Dimensions : environ 50 cm × 30 cm, profondeur 15 cm.
Présent depuis le passage de la tempête la semaine dernière.
Dangereux pour les véhicules, plusieurs voitures l'ont heurté.
```

### Exemple 2 : Éclairage

**Titre** : Lampadaire éteint parc des Roses

**Photos** : 2 photos (lampadaire de jour et environnement)

**Description** :
```
Le lampadaire situé à l'entrée est du parc des Roses
ne s'allume plus depuis environ 1 semaine.
Cette zone est maintenant très sombre la nuit,
ce qui pose un problème de sécurité pour les piétons.
```

### Exemple 3 : Propreté

**Titre** : Dépôt sauvage d'encombrants parking des Acacias

**Photos** : 2 photos montrant les encombrants

**Description** :
```
Plusieurs encombrants (matelas, meubles, cartons)
ont été déposés sur le parking des Acacias,
près des conteneurs de tri. Dépôt constaté ce matin.
Les déchets bloquent l'accès aux conteneurs.
```

## Questions fréquentes

**Combien de temps pour une réponse ?**
Généralement 24 à 48h pour la validation, puis quelques jours pour la résolution selon la complexité.

**Puis-je créer plusieurs signalements ?**
Oui, sans limite, tant qu'ils concernent des problèmes différents.

**Puis-je signaler un problème chez un particulier ?**
Non, uniquement les problèmes sur l'espace public (rues, parcs, bâtiments municipaux).

**Et si mon problème n'est pas résolu ?**
Vous pouvez ajouter un commentaire sur le signalement ou contacter directement les services municipaux.
