# Se connecter

Ce guide vous explique comment vous connecter à l'application Novaville.

## Première connexion

### 1. Télécharger l'application

Téléchargez Novaville depuis votre magasin d'applications :

- **iOS** : App Store
- **Android** : Google Play Store

### 2. Créer un compte

Au premier lancement :

1. Appuyez sur **"Créer un compte"**
2. Remplissez le formulaire :
   - **Prénom** et **Nom**
   - **Adresse email** (sera votre identifiant)
   - **Numéro de téléphone** (optionnel)
   - **Mot de passe** (minimum 8 caractères)
   - **Confirmation du mot de passe**
3. Acceptez les **Conditions d'utilisation**
4. Appuyez sur **"S'inscrire"**

### 3. Vérifier votre email

Un email de confirmation vous sera envoyé. Cliquez sur le lien dans l'email pour activer votre compte.

## Connexion standard

### Étapes

1. Ouvrez l'application Novaville
2. Sur l'écran de connexion, entrez :
   - Votre **adresse email**
   - Votre **mot de passe**
3. Appuyez sur **"Se connecter"**

### Rester connecté

Cochez l'option **"Rester connecté"** pour ne pas avoir à vous reconnecter à chaque ouverture de l'application.

> ⚠️ **Attention** : N'utilisez cette option que sur votre appareil personnel.

## Mot de passe oublié

Si vous avez oublié votre mot de passe :

1. Sur l'écran de connexion, appuyez sur **"Mot de passe oublié ?"**
2. Entrez votre **adresse email**
3. Appuyez sur **"Envoyer"**
4. Consultez vos emails et cliquez sur le lien de réinitialisation
5. Créez un **nouveau mot de passe**
6. Connectez-vous avec votre nouveau mot de passe

## Problèmes courants

### "Email ou mot de passe incorrect"

- Vérifiez que vous avez bien saisi votre email et mot de passe
- Vérifiez que la touche Majuscule n'est pas activée
- Essayez de réinitialiser votre mot de passe

### "Compte non activé"

Votre compte n'a pas encore été activé. Vérifiez vos emails (y compris les spams) pour trouver l'email de confirmation.

Si vous ne l'avez pas reçu :
1. Sur l'écran de connexion, appuyez sur **"Renvoyer l'email de confirmation"**
2. Entrez votre adresse email
3. Vérifiez vos emails

### "Trop de tentatives"

Pour des raisons de sécurité, après 5 tentatives échouées, votre compte est temporairement bloqué pendant 15 minutes.

Attendez 15 minutes puis réessayez, ou utilisez la fonction **"Mot de passe oublié"**.

### Problème de connexion Internet

Assurez-vous que :
- Votre appareil est connecté à Internet (Wi-Fi ou données mobiles)
- Votre connexion est stable
- Novaville n'est pas bloqué par un pare-feu

## Connexion avec authentification à deux facteurs (2FA)

Si vous avez activé l'authentification à deux facteurs :

1. Connectez-vous avec votre email et mot de passe
2. Entrez le **code à 6 chiffres** généré par votre application d'authentification (Google Authenticator, Authy, etc.)
3. Appuyez sur **"Vérifier"**

### Code de secours

Si vous n'avez pas accès à votre application d'authentification, utilisez un code de secours :

1. Appuyez sur **"Utiliser un code de secours"**
2. Entrez l'un de vos codes de secours
3. Appuyez sur **"Vérifier"**

> ⚠️ Chaque code de secours ne peut être utilisé qu'une seule fois.

## Sécurité

### Conseils pour un mot de passe sûr

- **Minimum 8 caractères** (12+ recommandé)
- Mélangez **lettres majuscules et minuscules**
- Incluez des **chiffres**
- Ajoutez des **caractères spéciaux** (@, #, $, etc.)
- N'utilisez **pas** d'informations personnelles (nom, date de naissance)
- Utilisez un **mot de passe unique** pour Novaville

### Protection de votre compte

- Ne partagez **jamais** votre mot de passe
- Déconnectez-vous sur les appareils partagés
- Activez l'**authentification à deux facteurs** (recommandé)
- Mettez à jour régulièrement votre mot de passe

## Déconnexion

Pour vous déconnecter :

1. Allez dans **Profil** (icône en bas à droite)
2. Appuyez sur **"Paramètres"** ⚙️
3. Faites défiler jusqu'en bas
4. Appuyez sur **"Se déconnecter"**
