# Utilisateurs

Cette section décrit les endpoints de gestion des utilisateurs. Ces opérations permettent d'administrer les comptes, de modifier les profils et de gérer les rôles au sein de la plateforme.

## Endpoints disponibles

- **[Liste des utilisateurs](./list)** : Récupérer la liste des utilisateurs enregistrés sur la plateforme (avec pagination et filtrage).
- **[Créer un utilisateur](./create)** : Enregistrer un nouvel utilisateur (citoyen, agent ou administrateur).
- **[Modifier un utilisateur](./update)** : Mettre à jour les informations d'un utilisateur existant.
- **[Supprimer un utilisateur](./delete)** : Supprimer définitivement le compte d'un utilisateur.
