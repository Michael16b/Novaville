# Authentification

Cette section regroupe les endpoints nécessaires à l'authentification des utilisateurs et à la gestion de la sécurité des sessions via les tokens JWT (JSON Web Tokens).

## Endpoints disponibles

- **[Login](./login)** : Authentification de l'utilisateur avec son adresse email et son mot de passe pour obtenir les tokens d'accès et de rafraîchissement.
- **[Refresh Token](./refresh-token)** : Obtention d'un nouveau token d'accès après expiration, en utilisant le token de rafraîchissement.
