# Mettre à jour un signalement

Mettre à jour un signalement existant.

## Endpoint

```http
PATCH /api/v1/reports/{id}/
PUT   /api/v1/reports/{id}/
```

## Authentification

Token d'accès (Bearer) requis.

## Notes

- Remplacer `{id}` par l'identifiant du signalement.
- Le payload et les droits seront détaillés ultérieurement.
