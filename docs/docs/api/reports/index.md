# Rapports

Cette section regroupe les endpoints liés aux signalements d'incidents urbains (anomalies, voirie, éclairage, etc.) par les citoyens, ainsi qu'au suivi de leur traitement par les services techniques.

## Endpoints disponibles

- **[Liste des rapports](./list)** : Consulter la liste des signalements effectués.
- **[Créer un rapport](./create)** : Soumettre un nouveau signalement d'incident.
- **[Modifier un rapport](./update)** : Mettre à jour le statut ou les informations d'un rapport.
- **[Supprimer un rapport](./delete)** : Supprimer un rapport d'incident.
