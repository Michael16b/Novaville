# Événements

Cette section détaille les endpoints de gestion des événements de la commune. Ces endpoints permettent de créer, lister, modifier ou supprimer des manifestations citoyennes.

## Endpoints disponibles

- **[Liste des événements](./list)** : Consulter l'ensemble des événements de la commune.
- **[Créer un événement](./create)** : Planifier un nouvel événement citoyen.
- **[Modifier un événement](./update)** : Mettre à jour les détails d'un événement existant (date, lieu, description).
- **[Supprimer un événement](./delete)** : Annuler et supprimer un événement.
