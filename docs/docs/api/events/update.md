# Mettre à jour un événement

Mettre à jour un événement existant.

## Endpoint

```http
PATCH /api/v1/events/{id}/
PUT   /api/v1/events/{id}/
```

## Authentification

Token d'accès (Bearer) requis.

## Notes

- Remplacer `{id}` par l'identifiant de l'événement.
- Le payload et les droits seront détaillés ultérieurement.
