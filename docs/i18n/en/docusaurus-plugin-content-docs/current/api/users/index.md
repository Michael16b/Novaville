# Users

This section describes the endpoints for user management. These operations allow administering accounts, updating user profiles, and managing roles.

## Available Endpoints

- **[List Users](./list)**: Retrieve the list of registered users (supports pagination and filtering).
- **[Create User](./create)**: Register a new user (citizen, agent, or administrator).
- **[Update User](./update)**: Update details of an existing user account.
- **[Delete User](./delete)**: Permanently delete a user account.
