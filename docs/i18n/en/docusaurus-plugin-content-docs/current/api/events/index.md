# Events

This section details the endpoints for municipal event management. These endpoints let you create, list, update, or delete citizen events.

## Available Endpoints

- **[List Events](./list)**: Get all scheduled municipal events.
- **[Create Event](./create)**: Plan a new municipal event.
- **[Update Event](./update)**: Update details of an existing event (date, venue, description).
- **[Delete Event](./delete)**: Cancel and remove an event.
