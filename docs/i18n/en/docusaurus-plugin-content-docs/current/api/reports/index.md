# Reports

This section groups endpoints related to reporting urban incidents (potholes, roads, lighting, etc.) by citizens, and tracking how technical services resolve them.

## Available Endpoints

- **[List Reports](./list)**: Get the list of submitted incident reports.
- **[Create Report](./create)**: Submit a new incident report.
- **[Update Report](./update)**: Update the status or info of a report.
- **[Delete Report](./delete)**: Remove an incident report.
